# Mentara 编程规约

## React

- 统一使用 prettier + eslinct 格式工具
- 异步机制均使用 try...catch...机制

## Springboot

- API 设计符合[RESTful 设计规范](./docs/api-guide.md)

## 数据库表设计

- 三少原则：表数量少、字段数量少、联合查询少

- 适度冗余：用空间换时间（高频查询字段可冗余）

- 读写分离：主库写 + 从库读

- 冷热分离：历史数据归档

## Python

- 命名遵从下划线模式

## C++

- 使用 google 编程规范
- clang-format 格式化
